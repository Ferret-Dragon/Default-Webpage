# Content
There are two different classes of content: 

### 'content'
<section id="content-section" class="content">
    <h2>Content Section</h2>
    <p>Your content here</p>
</section>

### 'content centered'
<section id="centered-section" class="content centered">
    <h2>Centered Section</h2>
    <p>Your content here</p>
</section>
